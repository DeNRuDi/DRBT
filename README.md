# ПРЕДОСТЕРЕЖЕНИЯ
Данная программа работает только на `Linux`-системах. Для запуска требуется пакет `l2ping` и пакет `pybluez`

Пакет `pybluez` можно установить с помощью команды: 

`pip3 install pybluez`
